# 📟 电路可靠性预测平台（Circuit Reliability Prediction Platform）



### **在线演示（Streamlit App）：https://circuit-reliability.streamlit.app/**



本项目面向**电子元器件与电路系统**的可靠性分析与演示，提供从“退化数据处理 → 物理模型参数拟合 → 元器件级可靠度/RUL → 电路级可靠性（串/并/混合）”的完整流程。

---

## ✨ 主要功能

### 1）数据选取与在线编辑
- 支持 **默认加载仓库内置 Excel 示例数据**，也支持用户**上传 Excel 覆盖示例数据**
- 支持工作簿（sheet）选择
- 支持表格在线编辑（AgGrid）

### 2）数据处理
- 四分位距（IQR）异常值处理
- 指数加权移动平均（EWMA）平滑降噪
- 支持导出预处理后的数据（CSV）

### 3）元器件退化模型参数拟合
- 提供多种退化模型选择（如肖特基、PN结、双极晶体管等）
- 支持 **正常数据/增量型数据** 两种拟合模式  
  - 增量型数据：每个时刻数据相对初始值（t=0）做差
- 采用 **SSA（Sparrow Search Algorithm, 麻雀搜索算法）** 进行参数全局搜索拟合
- 输出拟合参数、损失曲线、拟合效果图与置信区间

### 4）元器件可靠度与累计失效概率
- 基于寿命样本拟合对数正态分布（lognormal）
- 输出：
  - 可靠度函数 **R(t)**
  - 累计失效概率 **F(t)=1-R(t)**  
- 自动标注关键时间点（如 t(0.95) 等）

### 5）电路级可靠性求解（RBD / 串并联系统）
支持手动输入元器件可靠度 \(R_i(t)\)，计算电路级可靠性：
- 串联模型
- 并联模型
- 混合模型：串联若干 Stage，每个 Stage 可为单元件或并联组，并提供结构示意图与计算明细。
